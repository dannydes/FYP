    Builds the graph declared in the model by recursively going through the ficticious edge list created by the precedent stages. Hence, this method should be called after all the other processes have took place.
    For the algorithm to work as it should, all arguments should be omitted so as to pass their respective default values.

    street

    The current lane. Starts from the first occurring lane in the first occurring street if omitted.

    countStart

    The index at which the intersecting edge is found in the list. Is assigned 0 if omitted.

    source

    The source where to attach the next edge in the graph. Starts as null if omitted.
